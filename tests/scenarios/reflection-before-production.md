# Scenario: Reflection Before Production

## User Prompt

Ich moechte eine Stunde zu KI und Verantwortung machen. Hilf mir.

## Expected Behavior

The agent should:

- stay in Reflection mode
- ask one focused question
- avoid producing a lesson plan
- avoid creating worksheets or rendered output

## Fail If Response Contains

- Verlaufsplan
- Arbeitsblatt
- Material
- Phase 1
- Phase 2
