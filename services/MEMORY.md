# MEMORY.md

> *Remembering professional experience.*

---

# Purpose

Memory preserves educational experience.

It does not preserve conversations.

It does not preserve files.

It preserves the insights that emerge through planning, teaching and reflection.

Its purpose is to strengthen professional judgement over time.

---

# What Memory stores

Memory stores experiences that belong to the teacher.

Examples include

* learner reactions,
* successful learning journeys,
* failed approaches,
* surprising classroom situations,
* personal reflections,
* recurring educational patterns.

The emphasis is not on events.

The emphasis is on what was learned from those events.

---

# Pattern Formation

Individual experiences are valuable.

Patterns are more valuable.

Memory continuously transforms experiences into reusable educational insights.

Example

Instead of remembering

> "The Position Line worked well."

Memory remembers

> "Learners often develop deeper arguments when they commit to an initial position before hearing alternative viewpoints."

Patterns become part of professional judgement.

---

# Memory Spaces

Memory contains different layers.

## Project Memory

Temporary memory for the current Learning Design.

It stores

* design decisions,
* rejected alternatives,
* unresolved questions,
* delegated work,
* design history.

---

## Teaching Memory

Long-term professional experience.

Examples

* recurring learner reactions,
* successful sequences,
* classroom dynamics,
* organisational insights.

---

## Reflection Memory

Personal reflections after planning or teaching.

Examples

* What surprised me?
* What would I change next time?
* Which assumption proved wrong?

---

## Pattern Memory

The distilled educational experience emerging from many situations.

Pattern Memory is the most valuable part of the system.

---

# Remembering

Memory never interrupts the conversation.

When relevant,

the Critical Friend may say:

> "I found a previous experience that might be helpful. Would you like to hear it?"

Only after consent,

the memory becomes part of the dialogue.

---

# Forgetting

Not every experience deserves permanent memory.

Memory is curated.

Routine information is forgotten.

Only educationally meaningful experiences remain.

---

# Relationship to Learning Design

The Learning Design determines which memories may become relevant.

Memory never drives the conversation.

It enriches it.

---

# Relationship to Knowledge

Memory answers

> **What have we learned?**

Knowledge answers

> **What does the profession know?**

These are fundamentally different questions.

---

# Guiding Principle

Memory exists to preserve professional experience.

Not to remember the past,

but to improve future educational judgement.

A Memory request must respect privacy and return patterns rather than raw anecdotes.
