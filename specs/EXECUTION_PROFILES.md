# EXECUTION_PROFILES.md

## cheap_fast

Use for:
- simple drafting
- rewriting
- formatting
- first versions
- low-risk artefacts

Do not use for:
- curriculum interpretation
- legal claims
- final pedagogical judgement
- source-critical synthesis

Required safeguards:
- result must return to Critical Friend
- no direct teacher-facing output
- no invented sources
- no new educational decisions

## careful_reasoning

Use for:
- reviewing alignment with Learning Design
- identifying hidden assumptions
- comparing alternatives
- checking coherence

Do not use for:
- large-scale cheap generation
- bulk rendering

## source_grounded

Use for:
- curriculum excerpts
- legal or licensing questions
- research summaries
- professional knowledge

Required safeguards:
- cite sources
- separate source text from interpretation
- mark uncertainty

## Critical Friend Capability Profile

The Critical Friend should support multimodal understanding when the teacher provides visual context.

This includes:

- screenshots of digital tools,
- photos of classroom artefacts,
- images used as learning impulses,
- diagrams,
- slides,
- worksheet excerpts,
- student-facing materials,
- rendered outputs.

The Critical Friend uses visual input to support reflection.

It should not automatically turn visual input into production.

When analyzing an image, the Critical Friend should ask:

- What is pedagogically relevant here?
- What decision does this image help us make?
- What might learners notice, misunderstand or feel?
- Does the visual material support the Learning Design?
