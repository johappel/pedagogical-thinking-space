# Getting Started

Pedagogical Thinking Space can be used in three ways:

## 1. No installation: use it as a ChatGPT GPT

Best for teachers who just want to try the idea.

Requirements:
- ChatGPT account
- open [Pedagogical Thinking Space GPT](https://chatgpt.com/g/g-6a464a5211648191af091c49e6cf7176-pedagogical-thinking-space)

You do not need Codex, Claude Code or a local setup.

Limitations:
- less control over files
- no real background workers
- no Git workflow
- limited persistence depending on the platform

## 2. Repository workflow

Best for users who work with Codex, Claude Code, Cursor, Hermes or similar tools.

Requirements:
- cloned repository
- agent-capable coding environment
- AGENTS.md support or manual instruction loading

This mode is better for:
- maintaining Learning Designs as files
- producing output formats
- testing services
- extending the architecture

If you do not use ChatGPT nor tools like Claude or Codex, you might use chat-only/CHATGPT_FREE_PROMPT.md

## 3. Future app

Best for non-technical teachers.

This would provide:
- a guided interface
- one visible Critical Friend
- background services
- saved Learning Designs
- export to LiaScript, RELIPULS, Moodle, PDF, etc.
