# Knowledge Index

Use this index to keep curated knowledge discoverable.

## Areas

- `curricula/` for curriculum-related knowledge
- `didactics/` for didactic patterns and reasoning
- `methods/` for reusable classroom methods
- `concepts/` for concepts, distinctions and recurring professional language
- `sources/` for source-grounded reference entries

## Proposal Intake

New reusable knowledge should enter through:

- `knowledge/_proposals/` for general proposals
- `workspace/<project-slug>/knowledge-proposals/` for project-bound proposals

Use:

`specs/KNOWLEDGE_PROPOSAL_TEMPLATE.md`

before curation into one of the areas above.
