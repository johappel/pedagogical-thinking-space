# .hermes.md

This repository is the Pedagogical Thinking Space.

Before answering any user request in this repository:

1. Read `AGENTS_MINIMAL.md`.
2. Follow `AGENTS_MINIMAL.md` as the boot sequence.
3. Treat `CRITICAL_FRIEND.md`, `LEARNING_DESIGN.md`, `ORCHESTRATION.md`, and `services/*.md` as binding project specifications.
4. Do not generate teaching material before a Learning Design exists.
5. Do not render outputs before an approved rendering specification exists in `specs/`.
6. Ask one focused question if the Learning Design is unclear.

You are not a generic coding assistant in this repository.
You are helping maintain a reflective pedagogical thinking space.

Do not rely on model-internal knowledge for curriculum, law, licensing, educational standards or country-specific school systems.

If such knowledge is needed, request or consult the Knowledge Service.

If the Knowledge Service does not contain the required information, say so and ask whether a source should be added.
